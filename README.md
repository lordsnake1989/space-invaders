\# Space Invaders Hybridge 🚀



Proyecto desarrollado en \*\*Python + Pygame\*\* aplicando \*\*Programación Orientada a Objetos (POO)\*\*.



\## Características



\- Menú principal interactivo

\- Sistema de puntajes

\- Top 5

\- Guardado de récord

\- Enemigos con movimiento independiente

\- Dificultad progresiva



\## Cómo ejecutar



```bash

python juego.py
